# Выбор локации для скважины


## Данные

В наличии были данные с характеристиками нефти и ее запасах в трех регионах, по 10 000 проб в каждом.

## Задача

Требовалось построить модель машинного обучения, которая поможет определить регион, где добыча принесёт наибольшую прибыль.  

## Используемые библиотеки
*pandas*, *matplotlib*, *NumPy*, *Scikit-learn*.
